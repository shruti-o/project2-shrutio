Detailed Description

Simulates a dynamic load balancer for web servers.

The LoadBalancer distributes incoming requests across a pool of web servers, dynamically scaling the number of servers based on request queue size. It also blocks requests from restricted IP ranges and logs system state over time.

◆ Run()

void LoadBalancer::Run ( )

Runs the load balancer simulation.

Runs the main simulation loop for the load balancer.

Advances the simulation clock, assigns requests, scales servers dynamically, and logs state.

Each cycle:

  • Randomly adds new requests.
  • Processes requests on active servers.
  • Assigns queued requests to available servers.
  • Scales servers up or down if necessary.
  • Logs state to a file and prints summary every 50 cycles.

Dynamically scales the number of web servers.

Dynamically scales the number of servers based on the queue size.

Adds or removes servers based on queue size thresholds and enforces a cooldown between scaling events.

If the queue is too large, a new server is added. If the queue is too small, a server is removed, respecting a cooldown period to avoid frequent scaling.
