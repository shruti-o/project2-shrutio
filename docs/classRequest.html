Detailed Description

Represents a single request handled by the load balancer.

Represents a single request entering the load balancer.

A request contains source and destination IP addresses, a processing time measured in clock cycles, and a job type indicating whether the request is streaming or processing.

Each request includes source and destination IP addresses, a processing time measured in clock cycles, a job type (streaming or processing), and an arrival timestamp.
